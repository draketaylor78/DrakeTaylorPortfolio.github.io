The Legend of the Truck

Description

In this project I had to create a game in either Scratch or Python. This game was done in Scratch and involved the use of lists, variables, loops, and many other things.

Concepts Learned

This project was meant for us to put together everything that we have learned about Scratch to create a game. This involved the use of several variables to make keep track of the level and the amount of lives, loops to repeat the actions of enemies, and lists to tell the player what kind of environment the truck is in.
